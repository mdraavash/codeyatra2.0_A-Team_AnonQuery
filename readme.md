This is the github repo for Saral-Shikshya: A Smarter Education Solution 
This solution is presented to solve problems such as:
1. Class Interactions are minimal : Student have minimm interaction in class. This solution is introduced to provide an app where student are able to raise their question directly to their corresponding subject teachers. Here, each student shall be anonymous such that their is not shame in any types of questoin; small or big. This ensures that all student confusions are heard.
2. Teacher Evaluations : Student are able to fil up a feedback form for each class they attain where they can rate and review their teachers. This ensures that colleges are able to provide the most experienced and qualified teachers.
3. Drop Out Risk Management : This system is introduced with an AI system which will evaluate students performance and attendance and classify them in different classes - strong, average, and weak. Based upon the classification, teachers can provide special courses or monitor them accordingly.
4. Class Schedule Monitoring : This system is provided with an schedul monitoring system. Here, teachers will provide description about their avaliablity based upon which a routine can be formed by the administrator. Also in case of leaves, the system can inform the studnet about dismissal of their class and/or change in routine.
